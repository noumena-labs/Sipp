# 维护者指南

本指南面向基于 CogentLM 源码仓库进行开发的开发者，涵盖仓库结构、构建编排、测试规范、代码覆盖率要求和贡献流程。

如果只使用发布包，直接阅读[使用核心库](../packages/)。

## 目录与指引

- [源码构建](source-builds.md)：环境引导、`clm` 命令使用、运行源码示例和演示程序、编译各包目标产物。
- [架构](../architecture.md)：Crate 边界划分与包组织结构。
- [网关架构](../gateway/architecture.md)：网关系统分层设计。
- [测试](../testing.md)：内建的所有测试套件。
- [覆盖率](../coverage.md)：执行覆盖率测试及分析输出。
- [贡献](../contributing.md)：代码贡献规范与要求。
